##### 单精度
用科学计数法表示小数，转为二进制数后记作1.xxxx,存储时省去1.。
- 总长度32bit，即4byte。
- 1bit表示符号0正1负。
- 8bit表示指数(-127～128)，存储时为了不判断符号，加上127。
- 23bit表示尾数(小数)。
